# Customer Segmentation using K-Means Clustering

## Objective

This project groups customers based on their Annual Income and Spending Score using K-Means Clustering.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Machine Learning Algorithm

* K-Means Clustering

## Features Used

* Annual Income (k$)
* Spending Score (1-100)

## Output

The model segments customers into different clusters and visualizes them using graphs.
